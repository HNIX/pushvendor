Pushvendor

Rails Point of Sale

Download this project as a .zip file Download this project as a tar.gz file

PV is a Ruby on Rails point of sale. It is made to facilitate small businesses looking for a cost-effective point of sale solution. Small businesses pay too much for bloated and outdated proprietary systems. PushVendor is free, open source and easily integrated. The project is still early in development. I welcome collaboration and pull requests.

Default admin credentials: admin, user

Demo: demo.pushvendor.com
